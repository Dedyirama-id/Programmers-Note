  - Stack             : Undo

  - Sorting           : pengurutan notes, todo
  - Searching         : search notes, todo
  - Collision Handling: collision handling hash table
  - Tree              : sub notes, sub todo
  - Graph             : todo
  - Hash Table        : user account
